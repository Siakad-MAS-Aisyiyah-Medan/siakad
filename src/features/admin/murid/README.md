# Modul Data Murid

## Struktur
- `index.jsx` — halaman utama
- `hooks/useMurid.js` — state, fetch, promote, delete
- `services/murid.service.js` — API `/murid`
- `components/MuridTable.jsx` — tabel data
- `components/MuridFilter.jsx` — pencarian

## API
- `GET /murid` — list
- `PUT /murid/:id` — promosi/update role
- `DELETE /murid/:id` — hapus
